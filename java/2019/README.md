# API Security OWASP Top 10 POC (Java版)

本目录为Java实现，适用于仅有JDK 1.8环境的服务器。包含服务端和客户端，覆盖OWASP API Security Top 10。

## 目录结构
- server/  服务端（Spring Boot，10个API端点）
- client/  客户端（Java，发起Top10攻击/异常请求）

## 快速开始

### 1. 服务端
- 进入server目录，使用Maven或Gradle编译并运行

### 2. 客户端
- 进入client目录，编译并运行
